可以使用颜色修饰皮肤，也可以使用如下魔咒
| 咒语           | 释义   | 备注 |

| dark-skinned | 深色皮肤 |    |
| dark skinn   | 深色皮肤 |    |
| tan          | 皮肤晒痕 |    |
|              |      |    |
